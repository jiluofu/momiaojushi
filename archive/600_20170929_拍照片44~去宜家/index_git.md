
          
            
**2017.09.29**

本篇回顾一下冬季的一个周日，一起去宜家买东西。

出门时，天气寒冷，街道上人还非常少。




![](img/51001-d75c67db5c4682d0.jpg)




大风之后，天非常蓝。




![](img/51001-14bae97ffb1cbab7.jpg)




非要背着自己的小书包，戴着大帽子，这个帽子太小了，现在已经送人了。




![](img/51001-4419fd1e7ae0cf3d.jpg)




这些小店，都躲过了拆墙打洞的运动，一直到现在。




![](img/51001-0ca676a2a7e45e91.jpg)




到了停车场，去开车。




![](img/51001-26055578a6ca90b2.jpg)




到宜家了，从书包里要拿什么？




![](img/51001-e0f88cb49a0c0f02.jpg)




原来是米妮宝宝，早上特意装在书包里带来了。




![](img/51001-8728e04c511ede10.jpg)




今天来挑自己的小转椅。




![](img/51001-d165f2028f53e79e.jpg)




吃了一大份意面。




![](img/51001-e3eadad599f31d43.jpg)




越吃越香。




![](img/51001-a20ce7e270a07f31.jpg)

![](img/51001-c02abff184cdb1c7.jpg)




一个人报销一大盘。




![](img/51001-353f75ae8681db09.jpg)




开心地推着车。




![](img/51001-d278acc6ca465b67.jpg)




发现了个大鞋子，自己穿起来了。




![](img/51001-9b8d1f10a698b047.jpg)




放回原位。




![](img/51001-7284a9a7c2a6b461.jpg)




去货架上搬东西。




![](img/51001-dbec03b32dc292db.jpg)




去哪里结账？




![](img/51001-e1ad6d157a906022.jpg)




想自己搬下来？




![](img/51001-443b3a1af1f0a168.jpg)




准备走吧。

去玩儿秋千。




![](img/51001-bdf686a2d6649c55.jpg)
